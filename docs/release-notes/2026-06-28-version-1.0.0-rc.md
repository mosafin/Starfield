# Release Notes — Version 1.0.0 Release Candidate

**Date:** 2026-06-28  
**Tag:** `v1.0.0-rc`  
**App version:** `1.0.0`  
**Save version:** `2` (unchanged)

---

## Summary

Starfield Starmap Explorer v1.0.0 RC is the first production-ready release of the offline atlas and progress tracker. All planned Version 1.0 features are complete. This build focuses on stability, documentation, compatibility verification, performance baselines, and release packaging — **no new gameplay features**.

## What's included

| Area | Highlights |
|------|------------|
| **Core Atlas** | 120 systems, 558 planets, 63 locations |
| **Mission Atlas 2.0** | 122 missions, metadata, filters, cross-links |
| **Knowledge Atlas** | 45 validated entries |
| **Galactic Search** | Unified search, keyboard navigation |
| **Fleet Manager** | Ships, crew, homes per universe |
| **Route Planner** | Multi-hop paths, saved routes |
| **Compare Mode** | Up to 4 systems side-by-side |
| **Timeline** | Generated adventure history |
| **Insights** | Forecasts, heatmaps, recommendations |
| **DLC framework** | Shattered Space pack auto-loaded |
| **Polish** | Shortcuts, a11y, health report, responsive layout |
