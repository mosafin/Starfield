# Credits — Starfield Starmap Explorer v1.0.0

## Project

Starfield Starmap Explorer — offline atlas and progress tracker for *Starfield*.

## Game

*Starfield* © Bethesda Softworks. This is an unofficial fan project.

## Catalogue research

Mission chains, system names, planet bodies, and location POIs are compiled from
community research, in-game playthroughs, and public wikis. No proprietary game
assets are bundled.

## Software

Built as a single self-contained HTML application with:

- [Tailwind CSS](https://tailwindcss.com/) (CDN)
- [Font Awesome](https://fontawesome.com/) 6 (CDN)
- [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts (CDN)

## Contributors

Maintained through iterative development phases (22–40). See `CHANGELOG.md` and
`docs/release-notes/` for feature history.

## Player community

Thank you to Starfield players who reported bugs, validated catalogue entries,
and tested save migrations across universes and NG+ runs.
